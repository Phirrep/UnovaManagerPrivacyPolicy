# UnovaManagerPrivacyPolicy
Privacy Policy

Last Updated: 2/5/2026

1. Overview

This application is a locally installed tool used by a single eBay seller account to review, summarize, and analyze order activity, including orders associated with eBay Live sales streams.

This application is not publicly distributed and is used solely for internal seller operations.

2. Information Collected

This application accesses and stores certain information retrieved through authorized eBay APIs, including:

Order transaction data

Buyer usernames

Buyer names and shipping addresses

Order identifiers

Purchase quantities and item details

Stream-related sales metrics

This information is retrieved solely for order processing and sales analytics purposes.

3. How Information Is Used

Collected data is used exclusively to:

Combine and organize order records

Generate sales summaries and performance dashboards

Assist with fulfillment and order tracking

Provide stream-related sales analytics

The application does not sell, share, or distribute buyer information to third parties.

5. Account Deletion and Data Removal

If eBay notifies the application owner that a user has deleted their eBay account, reasonable efforts will be made to remove stored personal data associated with that user, in accordance with eBay Marketplace Account Deletion requirements.

6. Data Security

The application uses the following safeguards:

Secure eBay API authentication using OAuth tokens

Local system access controls

Use of encrypted HTTPS connections when communicating with eBay services

Access to stored data is limited to the seller operating the application.

7. Third-Party Access

This application does not provide buyer data to any third parties and does not use buyer data for advertising, marketing, or profiling.

This privacy policy may be updated if application functionality changes or if required by eBay policy or applicable privacy laws.
